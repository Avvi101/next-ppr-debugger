# Security Policy

## Supported Versions

| Version | Supported |
| --- | --- |
| 0.1.x | Yes |

## Reporting A Vulnerability

Please avoid filing public issues for security problems.

- Prefer GitHub private vulnerability reporting if it is enabled for this repository.
- Otherwise email `theworkersdesk@gmail.com` with the package name, affected version, reproduction steps, and any suggested fix.

I will acknowledge valid reports as quickly as possible and aim to provide a remediation plan or patch release promptly.
