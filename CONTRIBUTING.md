# Contributing

Thanks for helping improve `next-ppr-debugger`.

## Local Development

```bash
npm install
npm run test
npm run build
```

## Pull Requests

- Keep changes focused and documented.
- Add or update tests for behavior changes.
- Update the README or changelog when public behavior changes.

## Releases

- Bump the version with `npm version patch|minor|major`.
- Push the version commit and tag to GitHub.
- The release workflow is prepared for npm trusted publishing with provenance once the repository is connected as a trusted publisher in npm.
